Detailed Description

Time Properties Data Set per IEEE 1588-2019 Section 8.2.4.

Time properties from grandmaster clock with deterministic structure

Contains time metadata extracted from Announce message header flags and body. All fields map directly to IEEE 1588-2019 specification requirements.
