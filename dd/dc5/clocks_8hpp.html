Detailed Description

IEEE 1588-2019 PTP Clock State Machines Implementation.

Implements Ordinary Clock, Boundary Clock, and Transparent Clock state machines with deterministic design patterns as required by IEEE 1588-2019 Sections 9 and 10.

Note
This implementation follows OpenAvnu deterministic design principles:
  • No dynamic memory allocation in critical paths
  • No blocking calls or exceptions
  • Bounded execution time for all operations
  • POD types for hardware compatibility
