Detailed Description

PTP Port State Machine.

Implementation of IEEE 1588-2019 port state machine with deterministic state transitions and bounded execution time.

Implements state machine per IEEE 1588-2019 Figure 9-1 with:

  • Deterministic state transitions
  • Non-blocking message processing
  • Bounded memory usage
  • Hardware timestamping integration
