Detailed Description

IEEE 1588-2019 Annex I/J profile configuration parameters.

Defines complete profile configuration per IEEE 1588-2019 Annex I (Default and Power profiles) and Annex J (profile template). Each profile specifies timing parameters, delay mechanisms, and operational constraints for specific application domains.

See also
IEEE 1588-2019, Annex I.2 "Default PTP profile"
IEEE 1588-2019, Annex I.3 "Power profile"
IEEE 1588-2019, Annex J "PTP profile template"

ProfileResult IEEE::_1588::_2019::ProfileConfiguration::validate ( ) const
inline

Validate profile configuration parameters.

Returns
ProfileResult indicating validation success or failure

Validates:

  • Delay mechanism consistency (E2E uses delay_req, P2P uses pdelay_req)
  • Domain number range validity (0-127 per IEEE 1588-2019)
  • Message interval ranges per specification
  • Announce receipt timeout >= 2 per specification
